# Brumbaugh Detailing

A mobile auto detailing business website built for Brumbaugh Detailing, serving Arcanum, Ohio and surrounding areas.

🌐 **Live site:** [brumbaughdetailing.com](https://brumbaughdetailing.com)

## Features

- Services & pricing overview
- Booking system with SMS pre-fill for mobile users
- Before/after photo gallery
- Availability calendar
- Merch store

## Tech Stack

- HTML, CSS, JavaScript
- Deployed via Netlify
- DNS managed through Namecheap

## About

Built as a freelance project to give a local small business a professional online presence.
